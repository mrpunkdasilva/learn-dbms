2. CREATE - Criar Banco, Coleção e Documentos

Criar um Banco de Dados

use lab_crud

O MongoDB cria o banco quando o primeiro documento é inserido.

Criar uma Coleção

db.createCollection("clientes")

Se não existir, o MongoDB cria automaticamente ao inserir dados.

Inserir Documentos (INSERT)

// Inserir um documento db.clientes.insertOne({ nome: "João Silva", idade: 30, email: "joao@email.com", cidade: "São Paulo" }) // Inserir múltiplos documentos db.clientes.insertMany([ { nome: "Maria Souza", idade: 25, email: "maria@email.com", cidade: "Rio de Janeiro" }, { nome: "Carlos Oliveira", idade: 35, email: "carlos@email.com", cidade: "Belo Horizonte" } ])
