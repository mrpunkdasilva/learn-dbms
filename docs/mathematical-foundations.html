TEORIA_DOS_CONJUNTOS

Conceitos Fundamentais

┌────────────────────────┐ │ SET.FUNDAMENTALS │ ├────────────────────────┤ │ A ⊆ B: Subconjunto │ │ A ∪ B: União │ │ A ∩ B: Interseção │ │ A \ B: Diferença │ └────────────────────────┘

Operações Essenciais

  • União (A ∪ B): Todos os elementos de A e B

  • Interseção (A ∩ B): Elementos comuns entre A e B

  • Diferença (A \ B): Elementos em A que não estão em B

  • Complemento (A'): Todos os elementos que não estão em A

Aplicações em Databases

-- União de conjuntos SELECT * FROM table_a UNION SELECT * FROM table_b; -- Interseção SELECT a.* FROM table_a a INNER JOIN table_b b ON a.id = b.id; -- Diferença SELECT a.* FROM table_a a LEFT JOIN table_b b ON a.id = b.id WHERE b.id IS NULL;
