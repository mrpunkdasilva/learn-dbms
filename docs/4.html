4. UPDATE - Atualizar Documentos

Atualizar um Documento

// Atualizar o email do João Silva db.clientes.updateOne( { nome: "João Silva" }, { $set: { email: "joao.novo@email.com" } } )

Atualizar Múltiplos Documentos

// Adicionar um campo "ativo" para todos os clientes db.clientes.updateMany( {}, { $set: { ativo: true } } )
